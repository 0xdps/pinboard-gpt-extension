Features

Everything you need to organize ChatGPT conversations

Pin Messages & Chats

Hover over any message to pin it, or click "Pin Chat" to save the entire conversation.

🏷️

Tag & Organize

Add custom names and tags. Choose from 200+ preset tags or create your own. Quick search with fuzzy matching.

🔍

Search

Filter by text, name, or tags. Find what you need quickly.

⚡

Jump Back

Click to return to the original conversation with your message highlighted.

🔄

Sync

Pro users get automatic sync across devices using Chrome Sync or Firefox Sync.

📋

Chat Outline

Navigate long conversations. View and jump to any message with one click.
