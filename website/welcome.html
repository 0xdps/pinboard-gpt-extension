Quick Start Guide

Follow these simple steps to start organizing your ChatGPT messages

1

Visit ChatGPT

Go to chatgpt.com or chat.openai.com

The extension only works on official ChatGPT pages

2

Pin Individual Messages

Save specific responses you want to remember:

  • Hover: Move your mouse over any message → Click the pin button
  • A dialog will appear to add custom name, tags, and description
  • Click "Save" to add it to your Pinboard
3

Pin Entire Chats 🆕

Save complete conversations for future reference:

  • Look for the "Pin Chat" button (bottom-right corner)
  • Chat name auto-fills from sidebar
  • Description pre-filled with your first message
  • Perfect for saving entire project discussions or research threads
4

Navigate Long Conversations 🆕

Use Chat Outline for easy navigation:

  • Click "Chat Outline" button (bottom-left corner)
  • See all messages in the conversation at a glance
  • Jump to any message instantly
  • Great for lengthy technical discussions
5

Organize & Filter

Manage your pins efficiently:

  • Click extension icon to open your Pinboard
  • Use filter tabs: All / Chats / Messages
  • Search by name, content, or tags instantly
  • Click any pin to jump back to the original location

🔒 Privacy First

All your pins are stored locally on your device. No accounts, no cloud sync, no external servers. Your data is completely private.

🆕 What's New in v2.0

  • Pin Entire Chats: Save complete conversations, not just individual messages
  • Chat Outline: Navigate long conversations with an interactive message list
  • Filter Tabs: Quickly switch between viewing all pins, chats only, or messages only
  • Improved UI: Floating buttons for easy access without cluttering the interface

💡 Pro Tips

  • Pin entire chats: For project discussions or research sessions
  • Pin specific messages: For code snippets or important answers
  • Use Chat Outline: Navigate conversations with 50+ messages
  • Tag everything: Use up to 3 tags per pin for easy organization
  • Export regularly: Backup your pins to JSON files
  • Descriptive names: Make your pins easy to find later
