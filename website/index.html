How It Works

Three simple ways to pin messages

1

Hover Method

Move your mouse over any ChatGPT message → Click the "Pin" button that appears

2

Right-Click

Select any text → Right-click → Choose "Pin selection to GPT Pinboard"

3

Quick Pin

Click the "Pin Message" button in the bottom-right corner of ChatGPT

📦 GitHub Instructions

Quick Install from Source:

  1. Download from GitHub
  2. Go to chrome://extensions/
  3. Enable "Developer Mode"
  4. Click "Load unpacked" → Select folder
  5. Visit ChatGPT and start pinning!
